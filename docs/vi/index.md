# Tài liệu RateBox

Chào mừng bạn đến với tài liệu hướng dẫn RateBox! Tài liệu này sẽ giúp bạn hiểu, thiết lập và đóng góp vào nền tảng RateBox.

## RateBox là gì?

RateBox là nền tảng đánh giá sản phẩm và dịch vụ toàn diện, được xây dựng để cung cấp trải nghiệm người dùng tốt nhất cho cả người đánh giá và doanh nghiệp. Nền tảng này cho phép người dùng tạo, đọc và quản lý đánh giá cho các sản phẩm và dịch vụ khác nhau.

## Tính năng chính

- Frontend hiện đại với Next.js sử dụng Server Components và App Router
- Backend mạnh mẽ với Strapi và các plugin tùy chỉnh
- Cơ sở dữ liệu PostgreSQL để lưu trữ dữ liệu mạnh mẽ
- API toàn diện cho tất cả tính năng của nền tảng
