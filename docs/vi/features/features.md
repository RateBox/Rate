# Tính năng chính

## Quản lý sản phẩm

### Collection Types
- Products: Thông tin chi tiết về sản phẩm
- Categories: Phân loại sản phẩm
- Brands: Thương hiệu
- Tags: Gắn thẻ sản phẩm

### Thuộc tính sản phẩm
- Tên sản phẩm
- Mô tả
- Hình ảnh
- Giá cả
- Thông số kỹ thuật
- Đánh giá trung bình
- Số lượng đánh giá

## Đánh giá và bình luận

### Collection Types
- Reviews: Đánh giá sản phẩm
- Comments: Bình luận về đánh giá
- Reactions: Phản ứng với đánh giá/bình luận

### Chức năng
- Đánh giá sao (1-5)
- Upload hình ảnh
- Like/Dislike
- Báo cáo nội dung không phù hợp
- Sắp xếp theo mức độ hữu ích
- Lọc theo số sao

## Người dùng

### Collection Types
- Users: Thông tin người dùng
- Profiles: Hồ sơ chi tiết
- Follows: Theo dõi người dùng
- Bookmarks: Lưu sản phẩm/đánh giá

### Chức năng
- Đăng ký/đăng nhập
- Xác thực email
- Quên mật khẩu
- Cập nhật thông tin cá nhân
- Quản lý đánh giá
- Theo dõi người dùng khác

## Tìm kiếm và lọc

### Tìm kiếm
- Tìm theo tên sản phẩm
- Tìm theo thương hiệu
- Tìm theo danh mục
- Tìm theo tags
- Tìm trong đánh giá

### Bộ lọc
- Theo giá
- Theo đánh giá
- Theo thương hiệu
- Theo danh mục
- Theo thời gian

## Thống kê

### Dashboard
- Số lượng sản phẩm
- Số lượng đánh giá
- Số lượng người dùng
- Đánh giá trung bình

### Báo cáo
- Sản phẩm được đánh giá nhiều nhất
- Người dùng tích cực nhất
- Xu hướng đánh giá theo thời gian
- Phân tích sentiment

## API

### REST API
- CRUD operations cho tất cả collection types
- Filtering, Sorting, Pagination
- Authentication & Authorization
- Rate limiting
- Caching

### GraphQL API
- Queries cho tất cả collection types
- Mutations cho các thao tác CRUD
- Subscriptions cho real-time updates
- Custom resolvers
