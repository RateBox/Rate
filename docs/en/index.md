# RateBox Documentation

Welcome to the RateBox documentation! This guide will help you understand, set up, and contribute to the RateBox platform.

## What is RateBox?

RateBox is a comprehensive product and service review platform, built to provide the best user experience for both reviewers and businesses. This platform allows users to create, read, and manage reviews for various products and services.

## Key Features

- Modern Next.js frontend with Server Components and App Router
- Powerful Strapi backend with custom plugins
- PostgreSQL database for robust data storage
- Comprehensive API for all platform features
